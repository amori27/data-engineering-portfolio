# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-01

### Added

- Kafka producer generating realistic clickstream events
- Spark Structured Streaming job for 1-minute windowed aggregation
- FastAPI REST API with pagination and country filtering
- PostgreSQL schema with indexes for analytical queries
- Docker Compose for local infrastructure (Kafka, Zookeeper, PostgreSQL)
- Comprehensive test suite (unit + integration tests)
- CI/CD pipeline with ruff, mypy, pytest
- Full documentation with architecture diagrams
- Production-ready logging and error handling
- Support for environment variable configuration
